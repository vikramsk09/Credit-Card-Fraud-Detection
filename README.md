## 📌 Project Overview
This project is a Machine Learning-based Credit Card Fraud Detection system built using Python. The model analyzes transaction data to identify whether a transaction is genuine or fraudulent.

It demonstrates a complete end-to-end classification workflow, including data preprocessing, handling imbalanced data, visualization, model training, and evaluation.

---

## 🎯 Objective
- Detect fraudulent transactions using machine learning
- Handle imbalanced datasets effectively
- Gain hands-on experience with financial datasets

---

## 🛠️ Technologies & Libraries Used
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

---

## ⚙️ Machine Learning Workflow
1. **Data Collection**
   - Used a dataset containing credit card transaction records

2. **Data Preprocessing**
   - Handling missing values
   - Feature scaling (if applied)
   - Addressing class imbalance (if applicable)

3. **Data Visualization**
   - Used **Matplotlib** and **Seaborn** to analyze transaction patterns and fraud distribution

4. **Train-Test Split**
   - Used `train_test_split` to divide the dataset into training and testing sets

5. **Model Building**
   - Implemented **Logistic Regression** for classification

6. **Model Evaluation**
   - Evaluated performance using **accuracy_score** (and other metrics if applied)
